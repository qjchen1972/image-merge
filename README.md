A simple program based on  paper<< Object recognition from local scale-invariant features>> , The project provides some features:
* Image merge 
* image similarity 

Install
===
* pip install -r requirements.txt

Getting Started
====
* image merge
  python imgmerge -m 0
  
* Merge images under directory
  python imgmerge -m 2
  
* image  similarity  
  python imgmerge -m 1
